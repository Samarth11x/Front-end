This is a Header tag!
It is commonly used for the heading section and can be used in articles or sections.

This is the main tag!
It is used only once in the web page and a page must contain a main section with its header.

This is a section tag.
It can be used to format the main section of the body and provide extra information.
A section can be used multiple times in the main tag.
A section includes heading tags. Example:

Heading 1

Heading 2

This makes the section more structured and formatted.
This is an article tag!
It is commonly used to post any content or article that is contextual to the webpage and can be used multiple times.
An article can be used outside of the section tag as well as inside it. It is used to represent self-contained content.
This is aside tag and is not related or tangentily related to the content in the main section.
